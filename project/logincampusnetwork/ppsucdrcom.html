                            用于 PPSUC 校园网 Drcom 认证的 Windows 登录 / 下线工具。

                            本项目提供两个可执行程序：

                            • login.exe：登录校园网
                            • logout.exe：下线校园网

                            支持账号密码文件自动读取，也支持手动输入。

                            # 使用方法

                            # 1. 下载程序

                            在本仓库的 Releasesopen in new window 页面下载最新版本，解压后你会得到：

                            
login.exe
logout.exe
账号密码.txt

                            # 2. 配置账号密码

                            编辑同目录下的 账号密码.txt，内容格式如下：

                            
你的账号
你的密码

                            说明：

                            • 第一行是账号
                            • 第二行是密码
                            • 后面的内容会被忽略
                            • 文件编码建议使用 UTF-8（默认记事本即可）

                            # 3. 登录校园网

                            双击运行：

                            
login.exe

                            程序行为说明：

                            • 如果 账号密码.txt 存在且内容正确 → 自动登录
                            • 如果文件不存在或内容不完整 → 弹出窗口让你输入账号密码
                            • 登录成功或失败都会有弹窗提示

                            # 4. 下线校园网

                            双击运行：

                            
logout.exe

                            成功或失败都会有弹窗提示。

                            # 注意事项

                            • login.exe / logout.exe / 账号密码.txt 必须放在同一目录
                            • 修改账号密码 不需要重新下载或打包程序
                            • 本程序仅用于学习与个人网络连接管理，请遵守学校网络使用规定

                            # 技术说明

                            • Python + requests
                            • UI 使用 tkinter（Windows 原生）
                            • 使用 PyInstaller 打包为单文件 exe
                            • GitHub Actions 自动构建并发布
